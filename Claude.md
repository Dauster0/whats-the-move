This is a React Native / Expo app called Elsewhere.

KEY FILES:
- app/index.tsx — home screen, swipe deck
- app/concierge-detail.tsx — move detail screen
- services/suggestions.ts — GPT + API calls
- services/places.ts — Google Places API
- services/ticketmaster.ts — Ticketmaster API

DATA:
- User interests, energy, time stored in AsyncStorage
- Seen suggestions tracked in AsyncStorage
